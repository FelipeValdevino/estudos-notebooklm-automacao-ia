# Explorando o Poder do NotebookLM com IA

## 📚 Sobre o Projeto

Este projeto foi desenvolvido como atividade prática da DIO com o objetivo de explorar o uso do NotebookLM como ferramenta de apoio aos estudos utilizando Inteligência Artificial.

O tema escolhido foi:

### Automação de Processos com Inteligência Artificial

O objetivo principal foi compreender como a IA pode auxiliar na automação de tarefas, otimização de processos empresariais e geração de conhecimento através de prompts inteligentes.

---

# 🎯 Objetivos

- Estudar conceitos de automação inteligente
- Entender o funcionamento do RPA
- Aprender sobre Engenharia de Prompt
- Utilizar IA como apoio aos estudos
- Organizar conteúdos utilizando NotebookLM

# 📝 Mini Guia de Estudos

## O que é RPA?
RPA é uma tecnologia utilizada para automatizar tarefas repetitivas realizadas por humanos em sistemas digitais.

## O que é IA?
A Inteligência Artificial permite que máquinas simulem comportamentos humanos como análise, interpretação e tomada de decisão.

## O que é Engenharia de Prompt?
É a técnica de criar instruções mais eficientes para obter melhores respostas de modelos de IA.
